# Business Spec: Add Two Numbers

## Goal
Create a Python function `add(a, b)` that returns the sum of two numbers, with comprehensive unit tests.

## Requirements
1. Function `add(a, b)` in `state/add_numbers.py`
2. Tests in `state/test_add_numbers.py` using pytest
3. Handle integers, floats, and negative numbers
4. Tests must pass: `pytest state/test_add_numbers.py -v`

## Acceptance Criteria
- File `state/add_numbers.py` exists with `add()` function
- File `state/test_add_numbers.py` exists with 3+ test cases
- All tests pass
